# react-native-sample

Steps to build:

1. Run: yarn install 
2. Run: expo start

This project use this architecture:

src
 |--api
 |--context
 |--components
 |--hooks
 |--screens
 |--App.js
 
and the project use:
1. Axios for API requests
2. Reducers, Actions 
3. Context/Provider, in order to have the data available to anyone component 
4. Hooks, it is necesary for re-use code
